## Recurrence Features

- **Daily**: Repeat tasks every day
- **Weekly**: Select specific days of the week
- **Monthly**: Choose specific day of the month
- **Yearly**: Set annual reminders

## Tech Stack

- **Language**: Java
- **Database**: SQLite with custom helper
- **UI**: Material Design components (XML)
- **Architecture**: MVC pattern
- **Notifications**: AlarmManager + BroadcastReceiver
